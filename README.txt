Pico y Placa C# 

Problem:

Using object-oriented, tested code using the language that you feel most proficient in, please write a "pico y placa" predictor. The inputs should be a license plate number 
(the full number, not the last digit), a date (as a String), and a time, and the program will return whether or not that car can be on the road.edits



Envirnment:
	Microsoft Visual Studio 2010
	Microsoft .NET Framework 
	Versión 4.5.50938 SP1 Rel


Ejecutar la Aplicación:

	Copia el proyecto en tu máquina local, 
	Para ejecutar la aplicación debemos abrir el proyecto con microsoft visual studio 2010 e iniciar la depuración (F5),
	inmediatamente se iniciará la aplicación.

	Nota: 
	Se utilizó el patrón de diseño MVP (Model View Presenter) y la suit the pruebas unitarias de Microsoft

Para ejecutar las pruebas:

	Nos dirigimos al menú de Pruebas de visual studio, seleccionamos la pestaña ejecutar, luego todas las pruebas de la solución 
	de esta manera vamos a poder obser todos los resultados de las pruebas realizadas
